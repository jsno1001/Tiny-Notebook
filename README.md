# Tiny-Notebook
### Tiny-notebook is very small free text editor for linux systems and other on Electron

## Dowload project
``` bash
git clone https://github.com/jsno1001/Tiny-Notebook.git
cd Tiny-Notebook
npm install
```

## Start project
``` bash
npm start
```
